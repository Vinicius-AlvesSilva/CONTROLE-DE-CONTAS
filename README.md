# CONTROLE DE CONTAS:
Este projeto faz parte da **Atividade Extensionista: Finanças em equilíbrio com Excel e Visual Basic for Applications.xlsm- Trabalho Final** e tem como objetivo auxiliar no **controle de contas** por meio de um formulário desenvolvido em Excel com macros (arquivo `.xlsm`).

# Arquivos
- **Finanças em equilíbrio com Excel e Visual Basic for Applications.xlsm**: contém o UserForm para inserção e gerenciamento das informações.

# Como usar
1. Baixe o arquivo clicando em Code > Download ZIP ou clone o repositório:
git clone https://github.com/Vinicius-AlvesSilva/CONTROLE-DE-CONTAS.git
2. Extraia o arquivo utilizando WinRAR, 7-Zip ou outro programa de descompactação.

# Pode ser necessário desbloquear o arquivo:
.Clique com o botão direito no arquivo Excel.
.Vá em Propriedades.
.Na seção Segurança, clique em Desbloquear.
.Em seguida, clique em Aplicar e depois em OK.
.Abra o arquivo Formulario de controle de contas.xlsm no Excel e habilite as macros quando solicitado.
